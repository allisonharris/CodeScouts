Building a site from scratch

In this course, I'll teach you how to build a modern responsive one-page website using the latest techniques and processes. Together, we’ll use SCSS, responsive development and jQuery to build the very site that you’re looking at now. If you’re ready to learn web developement, then you’ve come to the right place. Follow my steps and together we’ll have a site up in no time!

Specifications

  • Instructor: Matt Steele
  • Length: 2 Hours
  • Technologies: CSS, HTML, jQuery
  • Level: Advanced Beginner
