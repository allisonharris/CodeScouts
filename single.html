#1 — Intro

I want to note that the techniques that I'm using on this project are just that: my techniques for this particular project. Likely, I'll use different methods on my next project, and then I'll adjust for the next one, and so on. The web changes everyday and developers also adjust to those changes. This is simply a way to build a project in the time that it was produced. It will become out of date, it's the nature of the beast. In fact, I look forward to rewatching these down the road and shaking my head and the *old* techniques we once used.:)
